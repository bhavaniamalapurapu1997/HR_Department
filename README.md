# HR_Department
This project analyzes and predicts employee attrition using multiple machine learning models, including Logistic Regression, Random Forest Classifier, XGBoost Classifier, and a TensorFlow-based Neural Network.
The goal is to identify employees at risk of leaving and understand the key factors influencing attrition, enabling organizations to take proactive retention measures.
Data Preprocessing: Cleaning, encoding categorical variables and feature scaling.
Exploratory Data Analysis (EDA): Visualizing trends, correlations, and attrition patterns.
Model Training & Evaluation: Logistic Regression, Random Forest Classifier, XGBoost Classifier, TensorFlow Neural Network
Performance Metrics: Confusion Matrix, Classification Report (Precision, Recall, F1-score),Accuracy Score
